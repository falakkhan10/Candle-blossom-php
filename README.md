# Candle Blossom — Full Stack E-Commerce Web App

##  Overview
Candle Blossom is a full-stack e-commerce platform developed for a handmade candle business. It includes complete frontend and backend functionality with admin control.

## Tech Stack
- PHP
- MySQL
- Bootstrap 5
- HTML, CSS
- WAMP Server

## Features
- User & Admin Authentication System
- Product Catalog with Search & Filters
- Shopping Cart & Checkout System
- Order Management (Pending → Confirmed → Delivered/Rejected)
- Admin Panel with CRUD Operations
- Wishlist & User Reviews
- Image Upload System
- Responsive UI (10+ pages)
